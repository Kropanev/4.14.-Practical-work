## Отслеживание файлов
---
**.gitignore** — текстовый файл, в котором задаются правила для исключения файлов из репозитория. Например:

```*.log
build/
temp-*
git ls-files --other --ignored --exclude-standard — список всех игнорируемых файлов.
```
